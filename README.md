# Personal Dashboard 🚀

An elegant, modern personal dashboard with smooth scrolling animations and real-time updates.

## Features

✨ **Modern Design**
- Smooth scrolling animations
- Gradient backgrounds
- Dark theme with accent colors
- Responsive layout

📊 **Dashboard**
- Real-time statistics
- Activity feed
- Project showcase
- Skills visualization

⚡ **Performance**
- Fast loading
- Smooth animations
- Real-time updates

## Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Design:** Tailwind CSS
- **Animations:** Intersection Observer API

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
python app.py
```

3. Open `http://localhost:5000` in your browser

## Features Explained

### Smooth Scrolling
- Parallax effect on hero section

### Animations
- Gradient text animations
- Card hover effects
- Progress bar animations
- Smooth transitions

### Real-time Updates
- Activity feed updates every 30 seconds
- Live statistics
- Responsive data fetching
